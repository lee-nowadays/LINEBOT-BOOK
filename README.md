# LINEBOT-BOOK

## 功能
主要是查詢讀墨READMOO網站上的輕小說內容

## 加入好友方式
- ID  
  1657157286
- QR code  
![image](https://user-images.githubusercontent.com/53147433/176470653-6aa6a4f9-0937-4602-9619-eb6cf20a5bcb.png)

## 指令
!new 查詢最新的10本書  
!hot 查詢30天的暢銷榜  

也可以直接輸入想查詢的書本，但不局限於輕小說  
例如:特殊傳說、炎炎消防隊  
當查詢的書沒有的時候會跳出不是查詢的書本請多加注意
